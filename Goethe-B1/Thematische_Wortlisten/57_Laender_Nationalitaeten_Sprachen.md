# Laender Nationalitaeten Sprachen

| Deutsches Wort | Einfache deutsche Erklärung / Synonym | English translation | Deutsche Phrase / Beispielsatz |
|---|---|---|---|
| Deutschland | Land in Mitteleuropa | Germany | Deutschland hat sechzehn Bundesländer. |
| Österreich | Land südlich von Deutschland | Austria | Wien ist die Hauptstadt von Österreich. |
| die Schweiz | Land mit mehreren Amtssprachen | Switzerland | In der Schweiz spricht man mehrere Sprachen. |
| deutsch | zur deutschen Sprache oder zu Deutschland gehörend | German | Ich lerne Deutsch. |
| österreichisch | zu Österreich gehörend | Austrian | Das ist eine österreichische Spezialität. |
| schweizerisch | zur Schweiz gehörend | Swiss | Sie lebt in einer schweizerischen Stadt. |
| die Nationalität | Zugehörigkeit zu einem Staat | nationality | Welche Nationalität haben Sie? |
| die Muttersprache | erste gelernte Sprache | native language | Meine Muttersprache ist Türkisch. |
| die Fremdsprache | zusätzlich gelernte Sprache | foreign language | Englisch ist eine Fremdsprache. |
| zweisprachig | zwei Sprachen verwendend | bilingual | Das Kind wächst zweisprachig auf. |
| international | mehrere Länder betreffend | international | Die Firma arbeitet international. |
| die Kultur | Traditionen, Kunst und Lebensweise einer Gesellschaft | culture | Ich interessiere mich für andere Kulturen. |
