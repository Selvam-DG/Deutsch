# Berufe

| Deutsches Wort | Einfache deutsche Erklärung / Synonym | English translation | Deutsche Phrase / Beispielsatz |
|---|---|---|---|
| der Arzt / die Ärztin | Person, die Krankheiten behandelt | doctor | Die Ärztin arbeitet im Krankenhaus. |
| der Lehrer / die Lehrerin | Person, die unterrichtet | teacher | Der Lehrer erklärt die Aufgabe. |
| der Ingenieur / die Ingenieurin | Fachperson für Technik | engineer | Die Ingenieurin plant eine Brücke. |
| der Verkäufer / die Verkäuferin | Person, die Waren verkauft | salesperson | Der Verkäufer berät die Kundin. |
| der Koch / die Köchin | Person, die beruflich Essen zubereitet | cook; chef | Die Köchin bereitet das Menü vor. |
| der Fahrer / die Fahrerin | Person, die ein Fahrzeug fährt | driver | Der Fahrer wartet vor dem Hotel. |
| der Journalist / die Journalistin | Person, die für Medien arbeitet | journalist | Die Journalistin schreibt einen Artikel. |
| der Mechaniker / die Mechanikerin | Person, die Maschinen repariert | mechanic | Der Mechaniker repariert das Auto. |
| der Polizist / die Polizistin | Mitglied der Polizei | police officer | Die Polizistin regelt den Verkehr. |
| der Künstler / die Künstlerin | Person, die Kunst schafft | artist | Der Künstler stellt neue Bilder aus. |
| der Programmierer / die Programmiererin | Person, die Software entwickelt | programmer | Die Programmiererin schreibt neuen Code. |
| der Handwerker / die Handwerkerin | Person mit praktischem Fachberuf | craftsperson | Der Handwerker repariert die Tür. |
