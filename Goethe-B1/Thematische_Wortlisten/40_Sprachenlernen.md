# Sprachenlernen

| Deutsches Wort | Einfache deutsche Erklärung / Synonym | English translation | Deutsche Phrase / Beispielsatz |
|---|---|---|---|
| die Sprache | System zur Verständigung | language | Deutsch ist eine wichtige Sprache. |
| die Muttersprache | erste Sprache eines Menschen | native language | Meine Muttersprache ist Arabisch. |
| die Fremdsprache | später gelernte Sprache | foreign language | Deutsch ist für mich eine Fremdsprache. |
| der Wortschatz | Gesamtheit der bekannten Wörter | vocabulary | Ich möchte meinen Wortschatz erweitern. |
| die Grammatik | Regeln einer Sprache | grammar | Die deutsche Grammatik ist manchmal schwierig. |
| die Aussprache | Art, wie Wörter gesprochen werden | pronunciation | Ich übe die Aussprache jeden Tag. |
| das Verb | Wort für Handlung oder Zustand | verb | „gehen“ ist ein Verb. |
| das Substantiv | Nomen; Wort für Person, Sache oder Idee | noun | Substantive schreibt man im Deutschen groß. |
| das Adjektiv | Wort, das eine Eigenschaft beschreibt | adjective | „schön“ ist ein Adjektiv. |
| der Satz | Gruppe von Wörtern mit vollständiger Aussage | sentence | Schreib bitte einen vollständigen Satz. |
| übersetzen | in eine andere Sprache übertragen | to translate | Kannst du diesen Satz übersetzen? |
| wiederholen | noch einmal sagen oder tun | to repeat | Bitte wiederholen Sie das Wort. |
