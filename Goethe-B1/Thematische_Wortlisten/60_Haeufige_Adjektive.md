# Haeufige Adjektive

| Deutsches Wort | Einfache deutsche Erklärung / Synonym | English translation | Deutsche Phrase / Beispielsatz |
|---|---|---|---|
| groß | von bedeutender Größe | big; tall | Das Haus ist sehr groß. |
| klein | von geringer Größe | small | Die Wohnung ist klein, aber gemütlich. |
| gut | positiv oder von hoher Qualität | good | Das Essen schmeckt gut. |
| schlecht | negativ oder von geringer Qualität | bad | Das Wetter ist schlecht. |
| neu | erst seit kurzer Zeit vorhanden | new | Ich habe ein neues Handy. |
| alt | seit langer Zeit vorhanden; nicht jung | old | Das Gebäude ist sehr alt. |
| leicht | nicht schwer; einfach | light; easy | Die Aufgabe ist leicht. |
| schwer | von großem Gewicht; schwierig | heavy; difficult | Der Koffer ist schwer. |
| schnell | mit hoher Geschwindigkeit | fast | Der Zug ist sehr schnell. |
| langsam | mit geringer Geschwindigkeit | slow | Bitte sprechen Sie langsam. |
| teuer | viel Geld kostend | expensive | Die Wohnung ist zu teuer. |
| billig | wenig Geld kostend | cheap | Das Angebot ist billig, aber gut. |
| wichtig | von großer Bedeutung | important | Gesundheit ist wichtig. |
| möglich | machbar oder denkbar | possible | Ist ein Termin am Freitag möglich? |
| notwendig | unbedingt erforderlich | necessary | Eine Anmeldung ist notwendig. |
