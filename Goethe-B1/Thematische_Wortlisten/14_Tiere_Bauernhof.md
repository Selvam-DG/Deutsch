# Tiere Bauernhof

| Deutsches Wort | Einfache deutsche Erklärung / Synonym | English translation | Deutsche Phrase / Beispielsatz |
|---|---|---|---|
| die Kuh | weibliches Rind | cow | Die Kuh gibt Milch. |
| der Stier | männliches Rind | bull | Der Stier steht auf der Weide. |
| das Kalb | junges Rind | calf | Das Kalb bleibt bei seiner Mutter. |
| das Schwein | Nutztier mit kurzem Rüssel | pig | Das Schwein wühlt im Boden. |
| das Schaf | Nutztier mit Wolle | sheep | Das Schaf steht auf der Wiese. |
| die Ziege | Nutztier mit Hörnern | goat | Die Ziege frisst Gras. |
| das Huhn | weiblicher Haushaltsvogel | hen; chicken | Das Huhn legt ein Ei. |
| der Hahn | männliches Huhn | rooster | Der Hahn kräht am Morgen. |
| die Ente | Wasservogel mit breitem Schnabel | duck | Die Ente schwimmt auf dem Teich. |
| die Gans | großer Wasservogel | goose | Die Gans läuft über den Hof. |
| das Pferd | großes Tier zum Reiten und Arbeiten | horse | Das Pferd zieht einen Wagen. |
| der Esel | graues Lasttier mit langen Ohren | donkey | Der Esel trägt schwere Säcke. |
