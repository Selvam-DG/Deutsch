# Moebel Haushalt

| Deutsches Wort | Einfache deutsche Erklärung / Synonym | English translation | Deutsche Phrase / Beispielsatz |
|---|---|---|---|
| der Tisch | Möbel mit Platte und Beinen | table | Das Essen steht auf dem Tisch. |
| der Stuhl | Sitzmöbel für eine Person | chair | Bitte nimm auf dem Stuhl Platz. |
| das Sofa | gepolstertes Sitzmöbel für mehrere Personen | sofa | Wir sitzen auf dem Sofa. |
| das Bett | Möbel zum Schlafen | bed | Das Bett ist sehr bequem. |
| der Schrank | Möbel mit Türen zum Aufbewahren | cupboard; wardrobe | Die Kleidung hängt im Schrank. |
| das Regal | Möbel mit offenen Böden | shelf; shelving unit | Die Bücher stehen im Regal. |
| die Lampe | Gerät, das Licht gibt | lamp | Die Lampe steht neben dem Bett. |
| der Spiegel | Fläche, die ein Bild zurückwirft | mirror | Der Spiegel hängt im Flur. |
| der Kühlschrank | Gerät zum Kühlen von Lebensmitteln | refrigerator | Die Milch steht im Kühlschrank. |
| die Waschmaschine | Gerät zum Waschen von Kleidung | washing machine | Die Waschmaschine läuft. |
| der Staubsauger | Gerät zum Entfernen von Staub | vacuum cleaner | Der Staubsauger ist im Schrank. |
| der Mülleimer | Behälter für Abfall | trash can; bin | Der Mülleimer ist voll. |
