# Computer Internet

| Deutsches Wort | Einfache deutsche Erklärung / Synonym | English translation | Deutsche Phrase / Beispielsatz |
|---|---|---|---|
| der Computer | elektronisches Gerät zur Datenverarbeitung | computer | Ich arbeite täglich am Computer. |
| der Bildschirm | Fläche zur Anzeige von Bildern und Text | screen | Der Bildschirm ist zu dunkel. |
| die Tastatur | Gerät mit Tasten zum Schreiben | keyboard | Die Tastatur ist kabellos. |
| die Maus | Gerät zur Steuerung des Zeigers | mouse | Klicke mit der Maus auf das Symbol. |
| das Programm | Software für bestimmte Aufgaben | program | Dieses Programm ist einfach zu bedienen. |
| die Datei | digital gespeicherte Information | file | Ich speichere die Datei auf dem Computer. |
| der Ordner | digitaler oder physischer Behälter | folder | Die Fotos sind in diesem Ordner. |
| das Passwort | geheimes Wort zum Anmelden | password | Ändere regelmäßig dein Passwort. |
| die Webseite | einzelne Seite im Internet | webpage | Die Webseite lädt langsam. |
| der Link | anklickbare Verbindung zu einer Seite | link | Schick mir bitte den Link. |
| herunterladen | Daten aus dem Internet auf ein Gerät kopieren | to download | Ich lade das Dokument herunter. |
| hochladen | Daten von einem Gerät ins Internet übertragen | to upload | Bitte lade das Foto hoch. |
